# GlossingMachine
Программа для типологического анализа языка по глоссированному тексту.
Что делает
Принимает на вход глоссированное предложение в формате Лейпцигской нотации и определяет три типологических параметра:

Порядок слов — SOV, SVO, VSO
Тип маркирования — эргативный, аккузативный, расщеплённая эргативность
Морфологический тип — агглютинативный, фузионный, изолирующий, полисинтетический

На основе трёх параметров программа выдаёт гипотезу о возможном ареале распространения языка и примеры похожих языков.
# Как запустить python main.py
Формат ввода

Введите Ваш текст:  
я-ERG-1SG собака-ABS видеть-PST-PFV

Введите перевод:    
я вижу собаку

Морфемы разделяются строго дефисом -, глоссы пишутся заглавными буквами по стандарту Лейпцига (ERG, ABS, NOM, ACC, PST и т.д.).
